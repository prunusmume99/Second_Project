## 📌 프로젝트 개요

### 🎯 목표
- 착석 감지 + 터치 + RFID 인증 → 루틴 시작 조건 판단
- 루틴 도중 착석 해제 시 → 일시정지
- 루틴 시간 기록 → MySQL 저장
- RGB LED 및 LCD로 상태 시각화

### 💡 주요 기능
- 사용자 인증 (RFID)
- FSR 센서 기반 착석 감지
- 루틴 시간 기록 및 저장
- 집중/휴식 구간 조명 전환
- 루틴 로그 LCD 출력

## 🧪 사용 장비

| 장비명 | 용도 |
|--------|------|
| Raspberry Pi 4 | 메인 제어 시스템 |
| FSR 센서 | 착석 감지 |
| 터치 센서 | 루틴 시작 입력 |
| PN532 RFID | 사용자 인증 |
| RGB LED (PCA9685) | 공부/휴식 조명 |
| LCD 16x2 | 루틴 상태 표시 |
| 기타 | 점퍼선, 브레드보드 등 |
